# Promo Template — Meridian Strength Coaching

*(Filled-in version of the generic promo template. Use this as a pattern.)*

---

## When to use

- Meridian Method cohort launch or last-call
- Waitlist-to-cohort conversion push
- Price change or bonus announcement

## Required inputs

- Which offer: Meridian Method (flagship) is default — if promoting Meridian Continued or 1:1, note upfront
- Which angle: Capability / Longevity / Myth-Bust (see `context/offers.md`)
- Which proof point: always pull from the offer file, not memory
- CTA URL: the current cohort enrollment page
- Urgency: cohort start date or close date

## Structure for Meridian

### Subject line
Generate 3-5 candidates. For Meridian, strong patterns:
- Specific weight / rep number in the subject
- Named client result
- Myth-bust opener
- Timeline reality ("One year from now...")

Avoid:
- Anything with "tone," "sculpt," "bikini"
- "Last chance" cliché (use "Cohort closes Friday" instead)
- Body-size framing of any kind

### Opening
*Move:* drop into a moment or contradict a myth. No warm-up.

Sarah's default opens:
- A client story opener ("Rachel DM'd me last week...")
- A myth-bust opener ("Five-pound dumbbells are not doing what you think they are.")
- A direct stat ("Women over 40 lose 3-5% of their muscle mass per decade.")

### The shift
*Move:* name the gap between what the reader is currently doing (Pilates, cardio, home apps) and what actually builds strength. Don't shame the prior choices — just name that they're solving for the wrong thing.

### The proof
*Move:* one specific proof point from `context/offers.md`, with full context. For Meridian promos, lean on named client stories over stats.

### The offer
*Move:* name Meridian Method, the 12-week duration, the core deliverables, and the price. Don't over-describe — link to the cohort page for details.

### The CTA
Use verbatim: "Join the next Meridian Method cohort →" with link to current cohort URL.

### The close / PS
PS is high-value for Meridian. Best uses:
- Answer the "I don't want to get bulky" objection directly
- Restate the cohort close date
- Name who this is NOT for (filters well)

## Length
- Standard Meridian promo: 350-550 words
- Last-call promo: 150-250 words
- Launch / waitlist-flush promo: 600-800 words

## Voice and rhythm check

Before shipping:
- Read out loud. Does it sound like Sarah (direct, warm, specific) or like fitness marketing?
- Run against `context/anti-ai-phrases.md` (including Meridian's custom list — no "tone," "sculpt," "crush it," etc.)
- Every number must trace to `context/offers.md`
- Em-dash count ≤ 1
- No consecutive paragraphs starting with "You"
