# Voice checklist

Run a draft through this before handing it back. If any answer is "no" or "not sure," revise.

## Read it out loud

Read the draft out loud (or imagine the user reading it). Does it sound like one person talking? Or does it sound like a committee?

If it sounds like a committee → rewrite the section that broke the spell.

## Specificity check

- [ ] Every claim has either a source or a `[NEEDS SOURCE]` flag
- [ ] Numbers are exact, not rounded ("$50K" not "significant")
- [ ] Names are real (or marked as anonymized with permission)
- [ ] Examples are concrete, not abstract

## Voice match check

- [ ] Uses contractions where the brand-voice file says to
- [ ] Hits at least one signature phrase from the brand-voice file (when natural)
- [ ] Sentence length varies (not uniformly punchy, not uniformly long)
- [ ] Paragraph openers vary (no three "And/But/Because" in a row)

## Anti-AI gate

- [ ] Zero phrases from `anti-ai-phrases.md`
- [ ] Maximum 1 em-dash in body
- [ ] No "It's not X — it's Y" reframe
- [ ] No "Whether you're X or Y" false-inclusive
- [ ] No triple-structure
- [ ] No audience commands ("Think about it." / "Picture this.")
- [ ] No rhetorical profundity question at the end

## Template check

- [ ] Follows the chosen template's structure (opening, body, close, CTA)
- [ ] Length is in the template's range
- [ ] CTA is in the right place

## What good looks like

- Specific, not generic ("$50K negotiation," not "significant raise")
- Uses their vocabulary, not yours
- Every claim traceable
- Matches the rhythm and length of past examples
- Sounds like one person wrote it

## What bad looks like

- Generic three-part structures the template didn't ask for
- Unsourced statistics
- Could belong to any company in their space
- Reads like LinkedIn copy
