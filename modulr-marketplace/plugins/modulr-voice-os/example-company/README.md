# Example Company: Meridian Strength Coaching

This folder shows what the `email-voice-os` system looks like when it's fully filled in. It's a fictional coaching business — **Meridian Strength Coaching** — that runs online strength programs for women 35-55.

Use this as a reference while filling in your own `context/`, `templates/`, `examples/`, and `briefs/` folders. Every file here has the same structure as the corresponding file in the main repo — just with real content instead of `{{FILL IN}}` slots.

## Why a fake company?

A real filled-in example answers the question "what am I supposed to write in these slots?" better than any set of instructions. Reading `context/brand-voice.md` from Meridian, you can see how specific the voice file should actually get.

## Why strength coaching for women 35-55?

Because it's different enough from email software, agencies, or consulting that you can see how the patterns apply across industries. If the system works for a strength coach, it'll work for you.

## Files in this folder

- `context/` — all five context files, fully filled in
- `templates/promo.md` — one template fully populated with Meridian's example structure
- `examples/` — two sample emails in the Meridian voice
- `briefs/example-campaign-brief.md` — a filled-in campaign brief

## What's intentionally simplified

- Only one offer (a real Meridian would have several)
- Only one template populated (a real Meridian would have 3-5)
- Only two examples (a real Meridian would have 10+)

The goal is to show the *shape* of a filled-in system, not to build an entire business.
